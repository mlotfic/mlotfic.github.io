## 13. Cable Chart & Report Form Elements

These elements define the structure of cable charts (wiring diagrams/interconnection tables) generated by EPLAN's report engine using the form-based layout system.

---

### `Form`
A report template defining the layout, fields, and formatting of a generated EPLAN output document (terminal diagram, cable list, parts list, PLC I/O list, cover sheet, etc.). Forms are created and edited in the **Form editor**.

**Implementation notes:**
- Forms use placeholder objects to bind to project data.
- Form structure: header area + data area (repeating rows) + footer area.
- Multiple forms can be assigned to the same report type for different output styles.
- Stored as `.f01` – `.f05` files or in the forms database.

---

### `Plot frame`
The drawing border/title block frame applied to all pages of a specified page type. Contains the outer border, title block fields (project name, document number, revision, date, approvals), and zone/grid markings.

**Implementation notes:**
- Defined as a separate EPLAN project page of type **Plot frame**.
- Assigned to schematic pages via **Page properties → Plot frame**.
- Plot frame properties reference project properties (auto-filled title block).
- Must match the page format (A4, A3, A2, A1, A0 or custom).

---

### `Cable chart data area external`
The repeating data rows section of a cable chart report that renders **external** cable information — cables leaving or entering the local panel from external sources.

---

### `Cable chart data area internal`
Repeating data rows showing **internal** cable/wire connections within the panel.

---

### `Cable chart data area via form position`
Data rows positioned at form-defined coordinates rather than flowing automatically. Enables custom-layout cable charts with non-standard column arrangements.

---

### `Cable chart header`
The column header row of the cable chart, displaying labels for each data column (Cable No., From, To, Core, Cross-section, etc.).

---

### `Cable chart header external`
Column header specific to the external cable section of a combined cable chart.

---

### `Cable chart header external: Counter target`
Sub-header showing the counter-target data for each external cable connection — the opposite-end connection point of the source device.

---

### `Cable chart header internal`
Column header for the internal wiring section.

---

### `Cable chart header internal: Counter target`
Counter-target sub-header for the internal cable section.

---

### `Cable chart header via form position`
Header placed at a custom form position.

---

### `Cable chart header: Counter target`
Counter-target header for the main cable chart section.

---

### `Connection / cable external`
An external connection or cable reference object in wiring/connection reports, representing connections that exit the panel boundary.

---

### `Connection / cable internal`
Internal wiring reference in reports.

---

### `Connection / cable via form position`
Connection/cable data object placed at a custom form coordinate.

---

### `Connection via form position`
A single wire connection rendered at a form-defined location in a custom report layout.

---

### `Connection, external`
External wire connection element shown in terminal or connection reports.

---

### `Connection, internal`
Internal wire connection element in reports.

---

### `Cable chart data area via connection point`
Data area content generated by navigating through connection points (terminal-by-terminal traversal) rather than cable-object traversal. Used for terminal strip-centric cable documentation.

---

### `Cable chart header via connection point`
Header for cable charts generated via connection point navigation.

---

### `Connection / cable via connection point`
Connection or cable data accessed via connection-point traversal logic in reports.

---

### `Connection via connection point`
Wire connection data accessed using connection-point navigation in reports.
