## 14. Pin End Elements (Connector Documentation)

Pin end elements document the individual contact ends within connector assemblies, forming the detailed wiring data for multi-pin connectors in harness and panel wiring documentation.

---

### `Female pin end`
The socket/receptacle contact end of a connector pin — the female-gender terminus of a wired connection at a multi-pin connector.

---

### `Female pin end: Cable chart data area`
Cable chart data rows associated with female pin end connections.

---

### `Female pin end: Connected PLC connection point`
The PLC I/O channel linked through a female pin end — used in I/O documentation for PLC connectors.

---

### `Female pin end: Connection`
The wire connection arriving at or departing from the female pin end.

---

### `Female pin end: Connection / cable`
The cable or connection associated with the female pin.

---

### `Female pin end: Pin`
The specific pin position/number of this female contact in the connector layout.

---

### `Female pin end: Pin part`
Catalog part for the female pin contact element (e.g., crimp socket, IDC contact, poke-home socket).

---

### `Female pin end: Pin part reference`
Instance reference for the female pin contact part, used for crimp-contact quantity calculations.

---

### `Female pin end: Plug part`
Catalog part for the female connector housing.

---

### `Female pin end: Plug part reference`
Instance reference for the female connector housing.

---

### `Female pin end: Target`
The wiring target as seen from the female pin end perspective in connection documentation.

---

### `Male pin end`
The plug/post contact end of a connector — the male-gender terminus. Structure mirrors Female pin end exactly:

| Sub-element | Description |
|---|---|
| `Male pin end: Cable chart data area` | Cable chart rows for male pin connections |
| `Male pin end: Connected PLC connection point` | PLC channel linked via male pin |
| `Male pin end: Connection` | Wire connection at the male pin |
| `Male pin end: Connection / cable` | Cable/connection at the male pin |
| `Male pin end: Pin` | Pin position/number |
| `Male pin end: Pin part` | Catalog part for the male contact element |
| `Male pin end: Pin part reference` | Instance reference for male contact part |
| `Male pin end: Plug part` | Catalog part for the male connector housing |
| `Male pin end: Plug part reference` | Instance reference for the male housing |
| `Male pin end: Target` | Wiring target from the male pin perspective |
